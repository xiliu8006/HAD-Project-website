## 📋 Project Overview

This is the official project website for HAD (Hallucination-Aware Diffusion Priors), showcasing a novel approach for enhancing sparse-view 3D reconstruction by addressing hallucination artifacts in diffusion-based priors.

**Purpose**: This website is designed for reviewers to conveniently view video results. Any images shown in the supplementary materials can be found as corresponding videos here for better visualization and comparison.
